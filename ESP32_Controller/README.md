# 追踪器（Tracker）

姿态追踪器和头部追踪器的代码，由于两者代码相似度极高，便集合一起公用一套代码，通过不同的设备ID进行区分，仅做数据采集的功能，具体的数据解析集中由Receiver处理。

## 配置

WIFI模式配置为STA模式，搜寻并连接至接收器网络。

- 名称：`RobotCmd_AP`
- 密码：`12345678`

具体的WIFI通信协议可在 `lib/protocol` 中查看。

`pin 4` 和 `pin 5` 配置为CAN通信引脚，接入CAN网络实现和IMU之间的通信。（也可接入其他CAN设备使用）

IMU选用的是达妙的IMU，非常便宜实惠，可以通过CAN总线快速实现通信。

## 备注

- 在做这个项目时，卡住最久的是达妙IMU的文档更新滞后，文档例程中提供的是和旧版固件的通信，和新固件并不适配。最后在各种排查、找资料才发现这个问题。
